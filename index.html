LIGGGHTS力链生成步骤

发表于 2020-12-09 阅读次数： Valine：

采用ParaView_Reader_for_LIGGGHTS

  1. 下载paraview4.3.1版本，该版本paraview不支持中文路径，会显示乱码

  2. 下载插件 libliggghts_forcechainreader.so

  3. 进入paraview4.3.1版本，点击Tools>Manage Plugin>Load New，在文件夹中选择libliggghts_forcechainreader.so

  4. 计算的in.liggghts中添加如下命令

  5. 在paraview中打开某个时刻的文件

  6. 选择Cell Data To Point Data过滤器

  7. 选择Calculator，输入公式$sqrt(F_X^2+F_Y^2+F_Z^2)$

  8. 选择Tube过滤器

  9. Tube中各个选项

    Scalars为计算出来的标量

    Vectors任意选择，不影响结果

    Number of Sides为圆柱的精细度

    Radius为最小圆柱的直径，可以调节到合适的大小来显示。

    Vary Radius选择By Scalar

    Radius Factor为最大值与最小值之间的倍数

    可以勾选Use Default Normal是圆柱更光滑
